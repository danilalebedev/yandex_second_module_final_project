## Описание проекта

Проект посвящён анализу пользовательского поведения в сервисе продажи билетов.  
Цель исследования — выявить факторы, влияющие на количество заказов и возврат пользователей.

В рамках работы:
- сформирован пользовательский профиль на основе истории заказов;
- проанализированы характеристики первого заказа;
- исследованы поведенческие и временные метрики пользователей;
- проведена сегментация аудитории по количеству заказов;
- выполнен корреляционный анализ признаков.

Результаты анализа позволили выделить ключевые драйверы повторных покупок и сформулировать рекомендации для роста пользовательской активности и удержания.
